# @julkwel/is-valid-date

## Installation
`npm install @julkwel/is-valid-date`


## Usage :
```
const isValidDate = require("@julkwel/is-valid-date");

isValidDate("2022-05-15"); // output true
isValidDate("22"); // output false
```

## Testing :
`npm test` or `yarn test`
